This is a project that explores thinking regarding skateboarding, and how people may think about it better.

Topics for Discussion

  • The manner with which skateboarders think about riding boards, executing/performing tricks, and utilizing objects
  • The physics/mechanics involved with the execution of tricks
  • The language that skateboarders use
  • The degree to which the language that skateboards use matches up with the concepts of physics
